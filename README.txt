# Password Generator Bot

Этот бот на Python использует библиотеку [aiogram](https://docs.aiogram.dev/en/latest/) для генерации паролей. Пользователи могут выбирать длину пароля, язык, регистр и дополнительные параметры, такие как использование цифр и специальных символов.

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше.

2. Склонируйте репозиторий:
   git clone https://github.com/ZXCKIRILL/-14.04-06.05

Установите необходимые зависимости:
pip install aiogram

Настройка
Получите токен для вашего бота через BotFather в Telegram.
Замените строку TOKEN = "ВАШ_ТОКЕН" в коде на ваш токен.

Запуск
Запустите бота с помощью команды:
python bot.py

Команды
/start - Приветственное сообщение и инструкции.
/help - Справка по командам.
/generate [длина] - Быстрая генерация сложного пароля (по умолчанию 12 символов).
/genpass <длина> - Создание пароля с настройками (длина от 6 до 50 символов).

Примеры использования
/genpass 12 - Начать процесс генерации пароля длиной 12 символов.
/generate 15 - Генерация случайного пароля длиной 15 символов.
